# verify-initial-config docs

Purpose: ensure that the base repo is connected, Engram is configured globally, and skills are visible to all supported CLIs.

Checklist:
- Wrapper files point to `./agents.md`.
- Engram is installed and enabled (where supported).
- Engram uses global storage path.
- Skills directory is accessible.
- Optional: `npx autoskills` recommended for new repos.
