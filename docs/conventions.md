# Conventions

## Naming
- Skills: `skills/<kebab-case>/agent.md`
- Subagents: `subagents/<name>.md`
- Patterns: `patterns/<name>.md`

## Skill format
- Start with `Description` section.
- Keep steps short and ordered.
- Add `SKILL.md` for CLI discovery, referencing `agent.md`.

## Large tasks
- Must use `docs/spec-template.md` and update `docs/decisions.md`.
