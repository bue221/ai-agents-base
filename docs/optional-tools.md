# Optional Tools

These tools are optional. Use them only when they improve the workflow.

## promptfoo
- Purpose: prompt testing, regression checks, model comparisons.
- Use when: you need repeatable prompt evaluations.

## Langfuse
- Purpose: observability, tracing, prompt versioning, evaluation.
- Use when: you need quality/cost/latency metrics.

## aider
- Purpose: pair-programming in terminal with git integration.
- Use when: multi-file or complex edits.

## llm CLI
- Purpose: multi-provider CLI, logging, fragments, templates.
- Use when: you want prompt libraries and local logs.

## fabric
- Purpose: reusable prompt patterns library.
- Use when: you want a curated pattern set.

## shell_gpt
- Purpose: fast shell command generation and log analysis.
- Use when: quick terminal assistance.
